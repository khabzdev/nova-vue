A modern Vue 3 component library built with [Reka UI](https://reka-ui.com) and [Tailwind CSS](https://tailwindcss.com).

## Features

- **Accessible** - Built on Reka UI primitives, ensuring WAI-ARIA compliance
- **Customizable** - Easily customize colors, sizes, and variants using Tailwind CSS
- **TypeScript** - Full TypeScript support with complete type definitions
- **Dark Mode** - Built-in dark mode support using CSS variables
- **Tree-shakeable** - Import only the components you need

## Installation

```bash
# npm
npm install @novavue/ui reka-ui

# bun
bun add @novavue/ui reka-ui

# pnpm
pnpm add @novavue/ui reka-ui
```

## Import Styles

```typescript
// main.ts
import '@novavue/ui/styles.css'
```

## Usage

```vue
<script setup lang="ts">
import { Button } from '@novavue/ui'
</script>

<template>
  <Button>Click me</Button>
</template>
```
